Main Thesis

ClassVision: YOWOv2-Based Student Activity Detection Framework

A privacy-conscious classroom activity-context framework for smart academic environments. The project uses YOWOv2 to classify visible person-level activity into studying, idle, collaborative, and arriving/leaving classes.

Problem

Room occupancy alone does not explain how a classroom is being used. A room may contain students who are studying, collaborating, waiting, entering, leaving, or sitting passively. The project addresses this by detecting activity context instead of only presence.

My Work

  • Organized classroom CCTV clips into a structured research dataset.
  • Prepared four activity labels and annotation rules for visible classroom behavior.
  • Produced label validation, cleaning, by-video data partitioning, and dataset summaries.
  • Prepared YOWOv2 fine-tuning workflow, inference outputs, post-processing logic, and demo materials.
  • Wrote manuscript sections, methodology notes, figures, and defense presentation assets.

Evidence

  • 108,116 processed label files.
  • 5,735 empty or no-person label files.
  • 1,951,112 annotated person-action boxes.
  • 72,491 training entries, 20,679 validation entries, and 14,661 test entries.
  • YOWOv2 medium, 16-frame clips, 224 image size, checkpoint saved at epoch 19.

Privacy Boundary

The framework avoids face recognition, identity matching, biometric profiling, and disciplinary automation. It reports activity context and room-level summaries instead of personal identity.
